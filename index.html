填寫規則

  • 每筆至少填寫一個 BBG Code；輸入美股 ticker 後會依交易所查詢檔自動補正，例如 AAPL → AAPL UW。
  • Barrier Type 為 EKI／AKI：Strike、KO Barrier、Coupon、Upfront / NotePrice、KI Barrier 五欄中，必須且只能空白一欄。
  • Barrier Type 為 NONE：KI Barrier 必須空白；Strike、KO Barrier、Coupon、Upfront / NotePrice 四欄中，必須且只能空白一欄。
  • Observation Frequency、OTC、Effective Date Offset 分別固定為 1、Note、7。
